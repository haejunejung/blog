## 해시 알고리즘
HASH란 해시 함수 혹은 해시 알고리즘을 사용하여 고정된 길이의 문자열로 변화시켜주는 것을 의미해요.

- 해시 함수: 임의의 길이의 데이터를 고정된 길이의 데이터로 매핑
- 키: 매핑 전 원래 데이터 값
- 해시 값: 매핑 후 데이터의 값
- 해싱: 매핑하는 과정
